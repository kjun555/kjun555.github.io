7.10 hexo博客

一.hexo博客的部署

搭建一个本地的网站

sudo apt install node nodejs-bin
sudo npm install hexo-clig
sudo apt install git
hexo init blog
cd blog
npm install
hexo server

二.hexo博客的更新

sudo apt install gedit
cd blog
hexo new “title”（日志系统)
hexo g
hexo d
hexo s

三.hexo博客的上线

上传到互联网上

在网站github注册一个账户创建一个地址
cd blog
sudo npm install hexo-deployer-git –save
git config –global user.email “2462345804@qq.com“
git config –global user.name “kjun555”
gedit _config.yml
hexo g
hexo d
