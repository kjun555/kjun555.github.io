Summer camp experience

fellings and experience

美好的时光总是短暂的，一眨眼六天的IT集训便过去了，在这短暂的时光中，对自己的专业有了更多的认识和了解，也见到了身边很多厉害优秀的人，很多努力的人，而且很荣幸有这么一位优秀的老师带我们学习。

身为一个计算机专业的学生，大学这两年感觉自己没有学到什么知识，虽然夏令营只有几天，但收获了很多，不仅仅是知识上，还有精神上和思想上，意识到了自己需要花更多的时间去学习更多专业知识，去提升自己.首先，开始通过虚拟机安装了四种操作系统，学习了一些系统语言和命令。通过hexo构成一个网站搭建，制作了一个日志系统，获益匪浅。其次，通过这几天的学习，让我深刻感受到自己的不足，IT行业需要有一定的英语水平，而自己的水平有时候无法理解一些英文，所以自己要花更多时间学习英文，以提高自己的英文水平。

所以，以后还会积极的参加一些训练和学习，只有不断学习，才能不断提升自己和完善自己，还要养成自觉学习的习惯，提高自身的知识水平和整体素质。

最后谢谢学校给予机会学习，谢谢杜老师的指导。

7.9 Some commands and functions

1.function

删除操作系统：点击上方虚拟机，点击管理，点击从磁盘中删除。（操作不可逆）

快照：文件拍摄快照之后，如果删除并且从回收站清空，可以通过恢复快照把文件还原。

2.Some commands

(1)系统级命令

命令 命令全称 命令含义 格式
mkdir make directory 创建目录 mkdir DIRNAME
ls list 显示目录中的文件 ls
touch touch 创建文件 touch FILENAME
rm remove 删除文件 rm FILENAME
cd change directory 切换目录 cd DIRNAME

(2)yum命令

命令 命令含义
yum list 显示软件列表
yum install APPNAME 安装指定软件
yum search APPNAME 搜索相关软件
yum check-update 检查是否需要更新
yum updte APPNAME 更新指定软件
yum update 更新所有软件
yum remove APPNAME 卸载指定软件

(3)命令的组成

命令=命令+选项(功能性扩展，可省)+参数(可省)

选项参数可以有多个，参数多个时要有空格。

选项和命令之间有空格，选项不要求顺序，可以合体。
